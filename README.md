## 项目结构
- **com.example.attendance**
    - **controller/** ：控制层，负责接收前端请求
    - **service/** ：业务层，负责处理核心业务逻辑
    - **impl/** ：业务层的具体实现类
    - **dao/** ：数据访问层，负责操作数据库
    - **entity/** ：实体层，对应数据库表
    - **util/** ：工具层，包含通用结果封装 (Result)
